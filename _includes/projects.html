An Enhanced Framework for Zero-Shot Reinforcement Learning

Lauriane Teyssier — Advisor: Zhang Ya-Qin
  • Motivation: Reduce computational cost and improve generalization in zero-shot RL agents
  • Methods: Enhanced Forward-Backward Representations, Neural Network (Transformer) improvements, Expectile Regression for regularization
  • Results: 9%–34% improvement over previous SOTA, reduced training time (17h → 11h)
